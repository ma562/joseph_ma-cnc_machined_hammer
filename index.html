About

The G-code was generated on Vericut which is a software program used to perform simulations for CNC machining. The code was initially written on a .txt file on Notepad which was then uploaded into the Vericut program.


The final CNC operation was done on the Hurco CNC machine and hammer head.
The engraving was done over an area of 2.5 * 0.6inches.
